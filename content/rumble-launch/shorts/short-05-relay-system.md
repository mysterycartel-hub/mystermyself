## Script

Watch how four AI agents pass work to each other — and nobody drops the ball.

This is the relay system. Claude writes the strategy. Breaks it into tasks. Drops a relay into Kiro's inbox. No Slack message. No email. Just a clean file handoff.

Kiro picks it up. Reads the instructions. Builds the feature. Pushes a branch. Creates a pull request. All without a single human keystroke.

Maurice? He never touches the terminal. He gives direction. He says GO. The system executes.

Think of it like a relay race. One runner finishes their leg, passes the baton clean. Next runner already moving. No fumble. No pause. No lost context.

Four agents. One flow. Strategy to code to branch to PR — smooth like a kitchen line on a Friday night rush. Every station knows its job. @mysterycartel built different.

That's the relay system. And it never drops the baton.
