# Trading Chef Remotion Starter

This is your beginner-friendly Remotion video kitchen.

## What it includes

- 1280x720 YouTube Daily Breakdown template
- 1080x1920 Shorts/Reels/TikTok template
- Trading Chef intro scene
- Safe text component to avoid cropping
- JSON-driven video data
- Placeholder chef avatar support

## Beginner Setup

1. Install Node.js LTS
2. Install VS Code
3. Unzip this folder
4. Open the folder in VS Code
5. Open Terminal in VS Code
6. Run:

```bash
npm install
```

7. Start Remotion:

```bash
npm run dev
```

## Add Your Chef Avatar

Place your PNG image here:

```text
src/assets/chef/chef-avatar.png
```

If you do not add one yet, the project shows a placeholder chef circle.

## Render Videos

Landscape:

```bash
npm run build:landscape
```

Vertical Short:

```bash
npm run build:short
```
