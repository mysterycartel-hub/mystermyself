# PRODUCT CANON — MysterMyself / Scott-King Coast

**Last updated:** June 23, 2026

---

## What Is MysterMyself?

MysterMyself is a **multi-income education and entertainment platform** built as a coastal city metaphor called the **Scott-King Coast**. Every income lane is a district. Every district is a destination. Every skill is a location you can visit and learn.

**Parent Brand:** MysterMyself / Scott-King Coast
**Domain:** mystermyself.com
**Newsletter:** The Opportunity List (via Beehiiv)

---

## Cross-Platform Systems

### Passport System
- **What it does:** User accounts, XP tracking, stamps, badges, progression
- **Auth method:** Supabase magic link (email OTP)
- **Status:** Built, needs Supabase env vars to activate on production

### XP & Progression Engine
- **What it does:** Awards XP for actions, unlocks content, tracks user journey
- **Status:** Built and integrated across districts

### Newsletter Funnel
- **What it does:** Captures leads → routes to Beehiiv → segments by interest
- **Newsletter name:** The Opportunity List
- **Status:** Active (needs API key in Vercel for server-side features)

### Payment System
- **What it does:** Stripe checkout for products (Gold Playbook, TCU Membership, etc.)
- **Status:** Built but intentionally disabled. Activate when first product is ready to sell.
