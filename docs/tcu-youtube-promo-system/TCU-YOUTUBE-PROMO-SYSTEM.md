## AUTOMATION PATH — How This Becomes Repeatable

**This workflow becomes a system by following this pattern:**

### Episode Template (Repeat weekly)
1. Pick 1 TCU character + 1 lesson from the Road Map
2. Paste the DeepSeek prompt template → get script + lyrics
3. Paste lyrics into Suno with style tags → get song
4. Paste image prompts into Dzine AI (swap character/lesson) → get 4-8 images
5. Upload images to Veo 3.1 → get 4-8 motion clips
6. Import clips + song into CapCut → assemble in 30 min
7. Export Long + Short versions
8. Upload to YouTube with SEO template → publish
9. Cross-post Short to TikTok, Instagram Reels

### What Can Be Templated:
- DeepSeek prompt (swap character name and lesson topic)
- Suno style tags (same every time)
- Image prompt structure (swap character, scene, lesson)
- Motion prompt structure (swap character, motion type)
- CapCut project template (same timeline structure)
- YouTube description template (swap episode details)
- Keyword tag base set (add 5 episode-specific tags)

| Step | Can Automate? | How |
|------|--------------|-----|
| Script generation | YES | Saved DeepSeek prompt template |
